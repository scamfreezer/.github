# Scamfreezer                                                                                                                                                                                                                                                                                                             
   
## Anti-scam tooling for the moments that matter.                                                                                                                                                                                                                                                                          
                                                          
A small portfolio of focused libraries for the in-browser steps of common scams — the moments where a scammer needs a victim's screen to lie convincingly, and a few lines of code can keep that lie from sticking.                                                                                                       
 
**Site:** [scamfreezer.com](https://scamfreezer.com) — live demo + the case for adoption                                                                                                                                                                                                                                  
**npm:** [`@scamfreezer/*`](https://www.npmjs.com/org/scamfreezer)
                                                                                                                                                                                                                                                                                                                          
---                                                       
                                                                                                                                                                                                                                                                                                                          
## Projects                                               

### [textfreezer](https://github.com/scamfreezer/textfreezer)
Sixty lines of JavaScript that prevent in-browser HTML edits, closing the doctored-bank-page step in refund scams.
